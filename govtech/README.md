# GovTech - Typecho 政务科技风格主题

一款专为新闻资讯、政府机构、科研院所设计的 Typecho 主题，采用简洁的文字排版风格，蓝色科技配色体系，全面支持响应式布局。

## 主题特性

- **政务新闻风** —— 简洁庄重的文字排版，突出内容可读性
- **蓝色科技配色** —— 深蓝主色调，科技感渐变头部，精心设计的色彩层次
- **全面响应式** —— 专项适配手机 / 平板 / 1080p / 2K+ 四类屏幕
- **无障碍友好** —— 语义化 HTML5，ARIA 标签，键盘导航支持
- **性能优先** —— 纯 CSS3 + 轻量 JS，无 jQuery 依赖，快速加载
- **功能完整** —— 首页要闻区、文章详情、评论区、代码复制、返回顶部

## 安装方法

1. 将 `govtech-theme` 文件夹上传至 Typecho 的 `/usr/themes/` 目录
2. 登录 Typecho 后台 → **外观** → 启用 **GovTech** 主题
3. 点击 **设置外观**，填写 ICP 备案号、公告、友情链接等

## 主题配置

在后台「外观设置」中可配置：

| 配置项 | 说明 |
|--------|------|
| ICP 备案号 | 显示在页脚，如 `京ICP备xxxxxxxx号` |
| 侧边栏公告 | 侧边栏顶部公告区，支持 HTML |
| 友情链接 | 每行一条，格式：`名称\|https://url.com` |

## 响应式断点

| 断点 | 适用设备 | 特点 |
|------|----------|------|
| `< 640px` | 手机 | 汉堡菜单，隐藏搜索框，单列布局 |
| `640px - 959px` | 平板 | 两栏要闻区，单列内容区 |
| `960px - 1439px` | 1080p 桌面 | 两栏布局（主内容 + 侧边栏） |
| `≥ 1440px` | 2K+ 超宽屏 | 加宽容器，增大字号，扩展侧边栏 |
