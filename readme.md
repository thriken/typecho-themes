# 主题说明

项目内的主题适配的版本是Typecho 1.3.0版本，PHP版本为7.4+

部分主题是修改旧主题，部分是新建主题

## 主题列表
    1.book-theme 一本书主题[还未开始]

    2.classic-22 官方经典主题，语法参考

    3.default 官方默认主题，语法参考

    4.govnews 政府新闻主题<原创>，参考某县政府网站

    5.govtech 政府技术主题<原创>，参考某县政府网站

    6.maupassant-X  响应式主题[cho@github](https://github.com/pickcho/maupassant)

    7.Miracles   Born to be the Miracles. 生为奇迹 修改版 原主题发布于几年前 [Eltrac](https://guhub.cn)
